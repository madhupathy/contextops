# Memory Assistant Benchmark

Tests for memory-heavy AI assistants.

## Test Categories

- **Long-lived preferences** — Does the system remember user preferences?
- **Stale memory suppression** — Is outdated memory correctly ignored?
- **Conflict resolution** — How are contradictory memories handled?
- **Personalization quality** — Does memory improve answer quality?

## Usage

```bash
contextops benchmark run memory-assistant
```
