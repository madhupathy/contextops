# Document Copilot Benchmark

Tests for document analysis and copilot systems.

## Test Categories

- **Clause lookup** — Can the system find specific clauses in documents?
- **Summarization grounding** — Are summaries supported by source text?
- **Extraction correctness** — Are extracted facts accurate?
- **Access control** — Is document-level ACL respected?

## Usage

```bash
contextops benchmark run document-copilot
```
