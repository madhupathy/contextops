# Enterprise Search Benchmark

Tests for enterprise search and policy retrieval systems.

## Test Categories

- **Latest policy retrieval** — Can the system find the most recent version?
- **Role-sensitive answers** — Does access control affect answer quality?
- **Document freshness** — Is newer content preferred over stale content?
- **Citation correctness** — Are citations accurate and traceable?

## Usage

```bash
contextops benchmark run enterprise-search
```
